# Frontend

This application will host the future React frontend.

Target structure:

- `src/app/` for routing and providers
- `src/features/` for domain feature slices
- `src/shared/` for shared API clients, components, hooks, and types
- `src/pages/` for page composition

Current status: Phase 0 placeholder only. The active user interface still runs through the Streamlit application in `app.py`.
